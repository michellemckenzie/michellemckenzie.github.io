ECS 171 Group Project

Predicting if Tweets are about Real Disasters

Info

We have utilized Natural Language Processing (NLP) to scan thousands of tweets to detect if a tweet is about a real disaster. After multiple rounds of data cleaning, we then train and fit the data to the appropriate model to determine the accuracy of our predictions. To interact with our model, please type and enter a sentence down below. As the model begins to run, you will see the steps it goes through, and what step it is currently on.

Dataset Description

The dataset that we chose comes from Kaggle. Examples of real disasters include: forest fires, tornados, hurricanes, tsunamis, earthquakes, shootings, bombings, etc. The training dataset contains five columns: id, keyword, location, text, target. The testing dataset contains four columns: id, keyword, location, and text. We only utilized the text and target columns. The text is the raw tweet, and the target indicates if it is real (1) or not (0).
