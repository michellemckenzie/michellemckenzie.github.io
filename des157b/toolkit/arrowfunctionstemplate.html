Random Number

My function asks for an integer, and will return a number that is randomized from a minimum value of 0, up to the integer that the user input. I looked at MDN documentation for a refresher on creating randomized integers.

Named Function

            function randomize(biggestNum){
                return Math.floor(Math.random() * biggestNum);
            }

Function Expression

            const randomize = function(biggestNum){
                return Math.floor(Math.random() * biggestNum);
            }

Arrow Function

            const randomize = biggestNum => Math.floor(Math.random() * biggestNum);
